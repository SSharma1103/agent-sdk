# Support

## Questions

Use GitHub Discussions when available:

https://github.com/SSharma1103/agent-sdk/discussions

If Discussions are not enabled, open a GitHub issue with the `question` label and include:

- what you are building
- SDK version or commit
- Node.js version
- minimal code snippet

## Bugs

Open a bug report with a minimal reproduction.

## Security

Do not report vulnerabilities in public issues. Follow [SECURITY.md](SECURITY.md).
